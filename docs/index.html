Project

The project goal is to manage multiple quizzes without development maintenance.
Every marketing member has been formed to the JSON structure (potentially using a web editor) so that they're completely independent in the user data form creation. The tool should therefore provide an automatic routing system and the form of free insertion into production or action of a developer.
